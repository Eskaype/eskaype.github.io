Thesis

Statistical Quantities to detect adversarial examples

Crafting adversarial examples have revealed a lot about the distribution of the trained model. It has been shown that these examples are non intuitively related to the data distribution. My graduate thesis focussed on one such problem ' Training a network to learn adversarial subspace from most (significant adversarial example + training set) such that it is robust over the adversarial subspace'. During course of research we aimed at answering several smaller problems to answer the existance of such examples. Part of my work was to explore statistical measures to compare the subspace/distribution of adversarial examples from the real data

In this exploratory work we aim to answer questions like :

  • Can the local curvature/landscape around the data point reveal information about how easily an image can be perturbed?
  • Can a subset of adversarial examples be shown to have most influence on network functions be provided for training instead of augmentting whole set?

    NashProp: An Approximate Nash Equilibrium Computation in Large Loopy Non Cooperative games

    Computing Nash equilibrium is exponential with the number of players. We explore Bayesian strategies to compute an approximate Nash equilibria. A Modified loopy belief propagation to compute an approximate Nash equilibria on multi player games was implemented as a part of this project. To solve a game using 'NashProp' algorithm We represent multi player 2 action games as graphs. Computation time of nash prop only depends on number of players directly influencing decisions of each player. In a graph this is the number of edges for each node. Hence local nash strategies and beliefs are passed from one Node to another. We find that Nash Prop worked better for Highly loopy graphs over graphs like in hotelling problem that only had one loop.

    Cognitive visualisations and Risk Evaluation in stocks

    Current algorithms for prediction and risk evaluation in portfolio management involve algorithms which rely on statsitical quantities that add noise in the prediction.As a part of this project we explored visualisation strategies to remove the dependence on variance.

  • 1. We used Hoefdings Inequality to compute the confidence of returns lying within a expected margin
  • 2. We explored visualisation techniques (differential yield vs differential confidence) to overlay good and bad stock on a single map. This enables us to spot a bad stock against several good stocks.
  • 3. We created realtime app using Yahoo API and Bokeh library
